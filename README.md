# RetainX
A Revenue Intelligence Platform

## Objective
To analyze transaction data and generate business insights.

## Dataset
Online Retail II dataset

## Project Status
Data Cleaning Phase

---

## Problem:
Businesses lose customers without knowing why.

## Solution:
Analyze customer behavior using RFM analysis and churn prediction.

## Tech Stack:
Python
Pandas
NumPy
Matplotlib
Scikit-learn

## Steps:
1 Data Cleaning
2 Feature Engineering
3 RFM Segmentation
4 Churn Prediction
5 Business Insights

Results:
• Identified high value customers
• Found churn risk segments
• Suggested retention strategies
